Zebra now owns several of the scanner brands such as Symbol.
The core scanner driver must be installed for the Zebra scanner driver.

For 32 bit Windows unzip Zebra_CoreScanner_Driver_(32bit)_v3.03.0013.zip in the w32 directory to a directory
of your choosing and execute Zebra_CoreScanner_Driver_(32bit)_v3.03.0013.exe within the directory. 

For 64 bit Windows unzip Zebra_CoreScanner_Driver_(64bit)_v3.03.0013.zip in the w64 directory to a directory
of your choosing and execute Zebra_CoreScanner_Driver_(64bit)_v3.03.0013.exe within the directory.
