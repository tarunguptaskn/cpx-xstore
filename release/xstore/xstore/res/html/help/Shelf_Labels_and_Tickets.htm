Shelf Labels and Tickets

You can print shelf labels for any item in the current item file. Shelf labels can be printed to a laser report printer if the report printer supports printing multiple labels per page, or to a label printer if using ZPL II (Zebra Programming Language) communication.

In addition to scanning/entering items in order to print labels, the system also provides an option to print shelf labels for all of the items on an entire receiving document at once, and from previously saved batches if using ZPL II communication.

As you enter item IDs, Oracle Retail Xstore Point of Service adds each item ID and number of labels to the View Port area of the screen. The item description and unit price are also shown. If the item you entered is associated with a pre-defined stock label type, the item is automatically added to the view port with this icon .

When selecting the stock label type, the Assign To All menu option will assign the selected Stock type to all items in the current list; however, this selection will not update the pre-defined stock type of the items and will not apply to items that are added after the Assign To All option is selected.

The following options are available before you print labels for the items:

  • Change Quantity: Select Change Quantity to change the number of labels to be printed.
  • Delete Item: Select Delete Item to remove an item from the list of item labels to be printed.

Note: There is no confirmation prompt before deleting the item. Be sure you are highlighting the correct item in the View Port display area.
  • Change Price: Select Change Price to change the price that is printed on the label.
  • Item Lookup: Select Item Lookup to look up an item using the item lookup search criteria form. The item can be added to the list of item labels to be printed.
  • Add Receiving Document: Select this option to print labels for all of the items on a receiving document.
  • Change Label [ZPL II Only]: Select Change Label to specify a different stock label type for the item.
  • Save Batch [ZPL II Only]: Select Save Batch to assign a name to the current list of items so it can be used again in the future.
  • Retrieve Batch [ZPL II Only]: Select Retrieve Batch to retrieve a list of previously saved batches.
