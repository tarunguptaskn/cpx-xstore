Begin Count

  1. Count the money in the till at the opening of the register:
    1. At the Count Summary screen, select the tender type you want to count and then select the Count Selected button to display the tender group’s count screen.
    2. Enter each value as required in the focus bar and press [Enter]. Oracle Retail Xstore Point of Service will highlight the next denomination in the list.
    3. Select Next Group to display the next count screen.
    4. Repeat steps a-c until all tenders are counted and then select Summary to return to the Count Summary screen.
    5. Review the amounts. If the totals are correct, select the Done Counting button.

  2. The system compares the opening amount to the previous close or set float amount:

    • If the amount matches, the system prompts to confirm the amount you entered and opens the register after you acknowledge the system amount.
    • If configured, the system prints a Begin Count receipt showing the amount of starting cash in the till and signature lines for the cashier and manager to sign confirming this open amount.
    • If the amounts do not match, the system displays a prompt to either recount the till or to accept the difference. Press [Y] to continue opening the register or press [N] to recount the till.

  1. If you counted the till from the Back Office, select the Register option to display the register screen.

The register is now open and a till has been counted and attached to the current register. The system may print a receipt indicating the register number that is open and the cashier ID who is at the register.
