Shipping Maintenance Options

The following options are available at the Shipping Maintenance screen before you ship the items and close the Shipping Document:

  • View Exceptions: Select this option to view a list of exceptions found in this Shipping Document. This list includes only those items where there is a difference between the number of expected items to be shipped and the actual counted items on the shipment. Items that are in balance are not included.

Note: For store-created documents, the expected item count is always zero. Only documents created by the home office and downloaded to the store show an expected quantity greater than zero.

  • The edit item options include changing the item quantity and voiding an item

Note: A Shipping Document that includes automatically created items (for transaction types such as work order repairs) cannot be changed using the Item Edit functions, unless your system is configured to allow overrides based on security privileges.

If you try to void a line item on a Shipping Document downloaded from the home office, Oracle Retail Xstore Point of Service displays a message indicating that the item cannot be voided. Press [Enter] to acknowledge the message and return to the Shipping Maintenance screen.

  • Item Lookup: Select this option to look up an item ID in the file. You also have the option to add the item to the Shipping Document if the Shipping Document was created at your store. However, you cannot add items to a Shipping Document created by the home office and downloaded to your store.
  • Edit Document: Select this option to change the Shipping Document address information.

Notes About Shipping Documents

  • The Document ID cannot be changed.
  • The Document Type cannot be changed unless the document was created at the local store.
  • Add Comment: Select this option to add comments for this Shipping Document.

The following options may only be available for Shipping Documents after you have created a packing list and printed a shipping label.

  • View Packing List: Select this option to view the packing list created for this Shipping Document.
  • Reprint Shipping Label: Select this option to print a shipping label for this Shipping Document.
