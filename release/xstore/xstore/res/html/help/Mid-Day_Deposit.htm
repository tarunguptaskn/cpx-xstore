Mid-Day Deposit

Mid-Day Deposit allows you to place funds into the store bank (safe) during the course of a working session. This activity is not necessarily restricted to a specific time, and may be used as often as necessary.

The Count Summary screen displays a list of tender groups that may be removed from the till and deposited in the store bank.

  1. Use the up and down arrow keys to scroll through the list and select a count group. Select the Count Selected option.
  2. Enter each value as required in the focus bar and press [Enter]. The tender type you select and your system’s configuration determine the type of count you must perform.  
  3. If additional funds are to be placed into the store bank, select the Next Group option.
  4. Repeat steps 2-3 until you've counted all tender groups to be deposited, select the Summary option to return to the Count Summary screen.
  5. Review the deposit amounts. If they are correct, select the Done Counting option. The system prints a receipt for the Mid-Day deposit.

Note: The difference between the Declared Amount (your count) and the System Amount is the amount remaining in the till.
