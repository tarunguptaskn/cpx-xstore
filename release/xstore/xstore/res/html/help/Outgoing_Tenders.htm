Outgoing Tenders

  1. Choose the outgoing tender type from the list and press [Enter].

Note: Only those types of tender that are configured to be given out appear in the list.
  1. When prompted, enter the amount that will be given to the customer and press [Enter]. The system opens the cash drawer.
  2. The system asks if the tender exchange is complete:
    • If you select Yes, the system prompts you to close the cash drawer. The tender exchange is now complete.
    • If you select No, and your system is configured to allow multiple tenders in an exchange, the system returns you to the outgoing tender list where you now have the option to add a new incoming tender to repeat the exchange process.

When the transaction is complete, the system prints a tender exchange receipt showing the incoming and outgoing tenders and amounts.
