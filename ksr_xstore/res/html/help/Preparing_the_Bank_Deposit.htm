Preparing the Bank Deposit

  1. Depending upon your system’s configuration and your store policy, you may be prompted to select the bank where the money should be deposited. Select a bank from the list and press [Enter].
  2. At the Bank Deposit Count Screen, count the tenders to be deposited in the bank.
  3. Select the Done Counting option when you have finished counting.
  4. The system displays a confirmation prompt showing the bank deposit tenders and amounts you counted. Select Yes to accept the bank deposit amounts as shown or select No to return to the Count Summary screen to make any adjustments as needed.

When the bank deposit count is complete, the system prints a deposit slip showing all tenders to be deposited and returns to the Store Bank Maintenance menu.
