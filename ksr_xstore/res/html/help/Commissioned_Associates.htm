Commissioned Associates

If your store tracks the associates’ sales for commissions and sales goals, you may be prompted to select a commissioned associate (or associates) to be linked to the sale transaction.

  1. If prompted, add a commissioned associate or associates to the sale transaction through one of two methods:

Select the sale associate or associates to be linked with this transaction from the list and press [Enter]. Use the spacebar to select multiple associates.

<OR>

Select the Select By Associate ID menu option to enter the associate’s ID and press [Enter].

  1. The system adds the associate to the sale transaction and the associates’ names are shown on the View Port.
