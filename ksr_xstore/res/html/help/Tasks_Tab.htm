Tasks

From the Register login screen, the Task tab displays the current day, open tasks for your store. To view and edit tasks assigned to you, select the My Tasks menu option, the Associate Tasks screen displays.  

About the My Tasks screen:

Associates with a Manager security level or higher can view the list of tasks for all employees, as well as the store tasks. Associates with any security level below the Manager level can only view tasks assigned to them or assigned to their group(s).

The My Tasks screen displays all tasks visible to you. The menu options available are based on the task status and the associated workflow.
