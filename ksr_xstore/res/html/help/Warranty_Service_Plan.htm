Warranty/Service Plan

Note: Warranty/Service Plans are only available for designated items as determined by the home office.

Items that are eligible for warranty plans but have not been associated with any plan show this No Warranty Purchased symbol on the View Port. The list of warranties available for this item will be listed. Select a warranty from the list if the customer would like to add a warranty plan, or press [Esc] at the warranty list if the customer does not want to add a warranty plan at this time.

Since a warranty/service plan must have an owner, you must associate a customer name with the plan. The plan owner may be a different person than the customer who purchases the item and the warranty/service plan. For example, the customer may purchase an item as a gift and also wants to assign the warranty to the person receiving the gift.

If the customer decides to purchase a warranty plan later, you have the following options on the Warranty menu:

  • Add Warranty - Select this option to add a warranty to a previously-purchased item.
  • Change Customer - Select this option to change the owner of the warranty when the warranty information is displayed on the View Port.

Adding a Warranty/Service Plan

To add a warranty to an item, select a source for locating a previously-purchased item that will be covered by a warranty/service plan:

 

  • Select Current Transaction if the item is in the current transaction. Use this option for customers who originally decline the warranty/service plan but change their minds before the transaction is tendered.

  • Select Customer History to locate the item in the customer’s purchase record. A customer must be associated with the transaction, otherwise, a message displays and indicates that no items are eligible for a warranty. Use this option if the customer does not have the original receipt, but the purchase record can be found in the purchase history for the customer.

  • Select Previous Transaction with Receipt if the customer has the original receipt for the item. The system retrieves the item information from the original receipt.

  • Select Manual Entry to enter the item information manually. Use this option if the customer does not have the original receipt and the purchase record cannot be found in the purchase history for the customer.

Changing the Warranty/Service Plan Customer

  1. Select the Change Customer option to change the owner of the warranty/service plan.

  2. The system displays a list of the warranties on the current transaction. Select the warranty/service plan you want to update and press [Enter].

  3. The system prompts for customer information. Enter the information for the new warranty/service plan owner as required by your store’s policy.

When the sale transaction is completed, the system prints a receipt with the warranty details.
