Changing Tenders

To change tendering amounts and delete tenders during the tendering process:

  1. If prompted asking if the sale is complete, press [N] for No to return to the Sale Tender list.
  2. Use the up and down arrow keys to select the Modify Tender option and then press [Enter].

  1. Select from the following options:
  • Select the Change Tender Amount option to change the amount of a tender. Select the tender to be changed from the list of tenders on the current transaction, then enter the new amount for the selected tender. You must enter a new tender amount greater than $0.
  • Select the Void Tender option to remove the tender type from the transaction. Choose the tender to be voided from the list of tenders on the current transaction and press [Enter]. Oracle Retail Xstore Point of Service removes the tender from the transaction.

Note: The color of the Focus Bar prompt indicates the transaction mode; green=sale mode, red=return mode.
  1. If the transaction still has a balance due, you must select another tender from the list of tenders to complete the transaction.
